# How To Import Choke-abo

## Dev Plugin Path

`Z:\ChokeAbo\ChokeAbo\bin\x64\Debug\ChokeAbo.dll`

## Add The Plugin In XIVLauncher

1. Launch FFXIV through XIVLauncher.
2. Open Dalamud settings with `/xlsettings`.
3. Go to `Experimental`.
4. Add `Z:\ChokeAbo\ChokeAbo\bin\x64\Debug\ChokeAbo.dll` to `Dev Plugin Locations`.
5. Open the plugin installer with `/xlplugins`.
6. Go to `Dev Tools > Installed Dev Plugins`.
7. Enable `Choke-abo`.
8. Run `/chokeabo` in game to open the main window.
9. Use `/chokeabo config` if you want the settings window directly.
